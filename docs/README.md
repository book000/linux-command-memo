# book000/memo

技術的なさまざまな分野に関するメモです。Linux 系や各種プログラミング言語、フレームワークなどの自分用メモ（備忘録）があります。

他の人が書いていないような内容であれば Zenn あたりに記事を書くのですが、すでに数十のブログ記事があるネタを毎回ググるのは嫌なのでリポジトリ内でメモを取ることにしました。
